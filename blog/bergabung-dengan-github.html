Bergabung Dengan GitHub

AlKoderi bergabung dengan GitHub dengan cara sebagai berikut:

  • Ke https://github.com/.
  • Klik Sign up.
  • Cantumkan email lalu klik Continue.
  • Cantumkan password lalu klik Continue.
  • Cantumkan username lalu klik Continue.
  • Ketik y untuk menjawab pertanyaan Would you... lalu klik Continue.
  • Selesaikan tantangan puzzle lalu klik Create account.
  • Cantumkan launch code yang diterima via email.
  • Selesaikan registrasi.
  • Klik Continue for free.
