Mengganti Foto Profil Di GitHub

AlKoderi mengganti foto profil di github dengan cara sebagai berikut:

  • Dari dasbord klik Settings.
  • Klik bagian foto profil.
  • Klik Edit pada Profile picture.
  • Pilih gambar profil.
  • Klik Set new profile picture.
