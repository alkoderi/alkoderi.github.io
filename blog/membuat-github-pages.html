Membuat GitHub Pages

AlKoderi membuat GitHub Pages dengan cara sebagai berikut:

  • Dari dasbor klik Create repository.
  • Cantumkan nama repositori pada kolom Repository name, contoh: alkoderi.github.io.
  • Cantumkan deskripsi repositori pada Description (optional).
  • Klik kotak Add a README file.
  • Klik Create repository.
  • Klik Add file lalu klik Create new file.
  • Cantumkan index.html.
  • Sebagai langkah awal silakan cantumkan yang dibawah ini.
  • Setelah selesai klik Commit new file.
